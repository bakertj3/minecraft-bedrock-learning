# Minecraft Bedrock Learning Projects

Parent learning Bedrock add-on development to help facilitate child's creative ideas.

## Projects
- HelloWorldPack - First learning project (texture modification)

## Development Setup
- **Platform:** Develop on Mac, test on Windows PC
- **Tools:** VS Code, Blockception extensions, GIMP

## Folder Structure
Each project follows standard Bedrock pack structure:
- `manifest.json` - Pack metadata
- `pack_icon.png` - Icon shown in Minecraft
- `textures/` - All texture files
- `sounds/` - All sound files
